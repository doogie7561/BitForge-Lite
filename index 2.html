BitForge Miner

Local browser proof-of-work miner using Bitcoin-style double SHA-256 hashing.

Idle

The app itself mines demo blocks.

This version performs local proof-of-work in your browser. It repeatedly changes a nonce, double-hashes the block data with SHA-256, and checks whether the hash meets the selected demo difficulty.

Important: this mines demo blocks only. It does real local hashing, but it does not connect to a Bitcoin pool, submit shares, receive rewards, or mine real BTC yet.
